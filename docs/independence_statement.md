## Independence-substitution measures
Recognising that single-author commercial-affiliation submissions carry
elevated perception risk for reviewers and editors, the following measures
are in place to support independent verification:

1. **Public archival.** The complete simulation framework, configuration
   files, raw and processed outputs, validation logs, and replication report
   are publicly archived at https://doi.org/10.5281/zenodo.19499806 with no
   embargo and no access restrictions.

2. **Deterministic execution.** All analyses are seeded with seed = 42 and
   `PYTHONHASHSEED = 0`. Re-execution from identical inputs produces
   identical outputs (verified at `reproducibility/replication_report.json`,
   absolute difference = 0 across all four key metrics under the specified seed).

3. **Cryptographic hash-chain.** The output manifest (`MANIFEST.sha256`)
   contains SHA-256 hashes of every file in the `outputs/` tree. Independent
   verifiers can recompute hashes and confirm bit-level reproduction.

4. **Versioned configuration.** The analysis plan (`analysis_plan.yaml`) has
   a registered hash (1fba2e24…) computed at the top of `scripts/run_all.py`.
   Changes to the plan would change the hash and break reproducibility.

5. **No proprietary data.** This study uses only synthetic data generated by
   the simulation framework. No real-world AI systems or patient data are
   used. There are no proprietary datasets that could be used to manipulate
   results in favour of any commercial outcome.

6. **AI-use disclosure.** Generative AI tools (Claude, Anthropic) were used
   for code drafting and refactoring, document formatting, and copy-editing
   of prose. They were not used to specify the SCM functional forms, to
   select parameter values, to design the comparator architecture, or to
   formulate interpretive claims; all such scientific decisions were made
   by the author.

7. **Methodological scope-bounding.** The paper does not propose specific
   threshold values for institutional adoption, does not endorse any commercial
   product, and does not derive any commercial recommendation from the
   simulation results.

## Reviewer-actionable verification
A reviewer or editor wishing to verify independence may:
- Download the public archive at the Zenodo DOI above
- Run `python reproduce_all.py` (Python 3.11; requirements.txt)
- Compare the regenerated outputs against `MANIFEST.sha256` (bit-level)
- Confirm the seeded determinism via `reproducibility/replication_report.json`
